# Krew index

This directory makes the kopiur repository double as a [krew custom
index](https://krew.sigs.k8s.io/docs/developer-guide/custom-indexes/):

```console
$ kubectl krew index add kopiur https://github.com/home-operations/kopiur.git
$ kubectl krew install kopiur/kopiur
```

`kopiur.yaml` is GENERATED by the release workflow (rendered from the
repo-root `.krew.yaml` template with the release tag + asset sha256s) and
committed here on every release — do not edit it by hand. Until the first
release that ships plugin binaries, no manifest is present and
`krew install kopiur/kopiur` will report the plugin as unknown.
